# contact-management-api
A lightweight Docker image for a RESTful Contact Management API built with Node.js + Express.

# Features
- REST API for managing contacts
- Node.js + Express backend
- Lightweight production-ready Docker image
- Supports environment-based configuration
- Includes validation, logging, and error handling

# Pull the Image
docker pull pornsakul/api-contact:latest
